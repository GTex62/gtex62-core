# Engine Docs

This directory is the canonical home for shared engine architecture, normalized
schemas, and future suite conversion references.

The current project name is `gtex62-conky-engine`, but the preferred future
identity is `gtex62-core`: the shared Lua/Conky foundation for gtex62 desktop
suites. See the rename roadmap for rationale, compatibility requirements, and
the staged migration plan.
